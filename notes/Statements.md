# if
General syntax:
```java
if (expression) {
	// Do something
}

if (expression) {
	// Do something
} else {
  // Do something else
}

if (expression) {
	// Do something
} else if (expression2) {
	// Do another thing
} else {
	// Do something else
}
```
# while
General syntax:
```java
while (expression) {
	// Do something
}
```
